These apps were built using modern web technologies with a focus on JavaScript, JSON, and GSAP. JavaScript served as the core programming language to create interactive and dynamic elements throughout the site.

Throughout the websites (apps), I utilized GSAP (a JavaScript library) to add animations and transitions.
In summary, these apps were built on a robust foundation of JavaScript, incorporating JSON for data management and GSAP for beautiful animations.
